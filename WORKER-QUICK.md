# 현장 작업자 — 빠른 안내 (1페이지)

> 실적만 입력하면 됩니다. 품목·계획·BOM 등록은 **사무실**에서 합니다.

---

## 접속 주소

**http://3.35.9.204/worker-input**

즐겨찾기에 넣어 두세요. (크롬·엣지)

---

## 입력 순서 (4단계)

```
① LOT 선택  →  ② 수량 입력  →  ③ 불량 상세  →  ④ 확인·저장
```

### ① LOT 선택

1. 주소 접속
2. 검색창에 **LOT 번호** 입력 (예: `LOT-001`) 또는 목록에서 카드 탭
3. 카드가 **노란색(선택)** 으로 바뀌면 **다음**

### ② 실적 수량

| 칸 | 의미 |
|----|------|
| **투입** | 이번에 작업한 수량 |
| **양품** | 정상 제품 |
| **불량** | 불량 (없으면 0) |

- 규칙: **양품 + 불량 ≤ 투입**
- **잔여 수량 한 번에 입력** 버튼이 보이면 LOT 남은 수량을 한 번에 넣을 수 있음
- **다음**

### ③ 불량 상세

- 불량이 **0**이면 이 단계는 **자동으로 건너뜀**
- 불량이 있으면: 유형별로 수량 입력 (합계 = 불량 수)
- **다음**

### ④ 확인·저장

1. 투입·양품·불량 숫자 확인
2. **실적 저장** 클릭
3. **「실적이 저장되었습니다」** 가 나오면 완료
4. **다음 실적 등록** 으로 이어서 입력 가능

## 자주 나오는 메시지

| 화면 메시지 | 의미 | 할 일 |
|-------------|------|--------|
| 진행 중인 LOT가 없습니다 | 입력할 LOT가 없음 | 사무실에 LOT 발행 요청 |
| 보류(HOLD) 작업지시 | 지시가 멈춘 상태 | 사무실에 지시 해제 요청 |
| 등록된 공정이 없습니다 | 공정 미등록 | 사무실에 MBOM 등록 요청 |
| 불량유형이 없습니다 | 불량 코드 없음 | 사무실에 불량유형 등록 요청 |
| 자재 부족 | 원자재 재고 부족 | 사무실에 자재 입고 요청 |
